# 🏋️ GymCore

Plataforma de gestión para gimnasios y centros fitness.

## ✨ Características

- Gestión de miembros
- Membresías
- Rutinas personalizadas
- Control de asistencia
- Pagos y reportes

## 🛠️ Tecnologías

- React
- Node.js
- Express
- PostgreSQL

## ⚙️ Instalación

git clone https://github.com/davidpontevega/GymCore.git  
cd GymCore  
npm install  

## ▶️ Ejecución

npm run dev  

## 🔑 Variables de entorno

DATABASE_URL=  
JWT_SECRET=  
PORT=3000  

## 📡 Endpoints

GET /members  
POST /membership  
GET /attendance
